# 맛 모델 검증 리포트 (Taste Model Validation)

> 전자혀 앱의 맛 과학 가정을 요리과학·감각과학 1차 문헌으로 교차검증한 결과.
> 방법: 다중 출처 딥리서치(6개 검색 각도 → 25개 1차/2차 출처 → 90개 주장 추출 → 25개 핵심 주장 3표 적대적 검증 → 21 확정 / 4 기각).
> 작성: 2026-06 · 검증 엔진 버전: deliciousness v2 + 계산엔진

## 한 줄 결론
**방향성은 1차 문헌으로 탄탄히 검증됨. 단, 정량 '배수·계수'는 과단순화 — 고정 상수 대신 농도의존 공식이 정석.** 핵심 메커니즘(감칠 시너지, 단맛 우세, 혼합 억제, 짠맛=풍미 베이스, 감칠↔소금 절감, 매운맛=별도 통각축)은 모두 사실로 확인.

|---|---|---|---|
| 1 | 감칠맛 시너지(글루탐산×이노신산) 존재 | ✅ **맞음** | T1R1/T1R3 수용체 알로스테릭 공동결합. 단독으론 무미, 함께면 초가산 (PNAS 2008, Sci Rep 2020, J Nutr 2022) |

| 3 | 혼합 총강도 < 성분 합 (억제) | ✅ **맞음**(방향) / ⚠️ '~50%' 수치는 미검증 | 혼합은 **하가산**(perceptual additivity가 최적 예측, 단순합은 과대추정). 단, 3성분 이상으로의 일반화는 약함(1-2 기각) |

| 8 | 장류 염도 ~10~19% w/w | ✅ **맞음** | 발효 콩장 측정 **9.7~19.3% 소금(w/w)**. 완성요리 목표(0.6~1.3%)의 ~10배 → 소량만 투입해야 |

## 기각된 주장 4건 (앱에 중요)
1. **혼합 억제의 3성분+ 일반화** — 이항(2개) 쌍에선 견고하나 3·4성분 일반화는 약함 (1-2).
2. **캡사이신의 짠맛/감칠 마스킹** — 사람 매운국 실험서 미발견 (0-3).
3. **E-tongue 감칠 = 글루탐산** — 실제론 **페닐알라닌**이 감칠 점수와 최상관 → 글루탐산 단독축은 불완전 (0-3).
4. **된장 감칠 = 자유 글루탐산** — 상당 부분 **저분자 산성 펩타이드**가 담당 (1-2).

## 앱에 반영한 보정
- **된장 염도 0.12 → 0.14**, 감칠 4 → 5: 전통 된장 18~20%·발효 펩타이드 감칠 반영(과거값은 상업 된장 하단에 치우침).
- (이전 커밋) **산미 비중 축소** — 항목 7의 "마무리 액센트" 원칙과 일치하게 이미 반영됨.

## v2 권고(엔진 개선 백로그)
1. **감칠 시너지를 곱셈 공식으로** — 고정 ×1.5 대신 `effectiveUmami = u·(1+γ·v)`, IMP/GMP 구분(γ_IMP≈1.2, γ_GMP≈2.8). 현재의 보수적 ×1.5는 *실제 음식 맥락*에선 방어 가능하나, 두 강한 감칠원 동시 사용 시 더 줄 여지.
2. **감칠↔소금 절감 트레이드오프** — 감칠 높으면 저염 페널티(salt<3) 완화. RCT로 강하게 뒷받침(0.9→0.3% 가능).
3. **짠맛을 '쓴맛 필터'로 모델** — 균일 증강이 아니라 쓴맛 함유 매트릭스에서만 단맛/풍미 부각.
4. **매운맛→타미각 마스킹 약화** — 사람 데이터선 미발견. 매운맛 페널티는 '마스킹'이 아닌 '균형/압도'로 한정.
5. **된장·간장 분리 벡터** — 서로 다른 대사체 프로필. 된장에 펩타이드 감칠 보너스.

## 한계 (정직한 단서)
- 대부분 근거는 **등강도 수용액·설치류 전기생리** — 실제 다재료·따뜻한 한식·한국인 입맛으로의 전이는 외삽.
- E-tongue은 기기 프록시(사람 관능 아님), 소표본(n=12 간장 등).
- **항목 6(요리별 적정 염도 0.6~1.3%)·7(산미 메커니즘)·10(규칙기반 모델의 한계: bliss point·개인/문화 편차)** 은 이번 검증셋에서 직접 확정되지 않음 — 별도 1차 근거 필요(앱은 식약처 권고 염도표로 보강).
